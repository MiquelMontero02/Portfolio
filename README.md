# Portfolio

Portfolio personal de Miquel Àngel Montero Pazmiño - Ingeniero Informático.

## Stack

Vue 3 + TypeScript, Vue Router, SCSS.

## Filosofía

Estilo visual Cyberpunk Arcade: fondos oscuros, colores neón cyan y pink, efectos glow en bordes y texto, tipografía pixel para títulos. Los datos del CV se cargan desde archivos JSON locales para mantener la app independiente.

## Ejecución

npm install
npm run dev
